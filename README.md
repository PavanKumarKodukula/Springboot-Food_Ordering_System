# Spring Boot Food Ordering System

## Overview
A backend Food Ordering Management System built using Spring Boot and MongoDB.  
This project exposes RESTful APIs and follows a layered architecture pattern (Controller → Service → Repository).

## Tech Stack
- Java
- Spring Boot
- MongoDB
- Maven
- REST APIs

## Features
- User Management
- Restaurant Management
- Food Item Management
- Order Creation and Tracking
- CRUD Operations
- MongoDB Integration
- Clean Layered Architecture

## Project Architecture
The application follows a structured layered design:
- Controller Layer – Handles HTTP requests
- Service Layer – Contains business logic
- Repository Layer – Communicates with MongoDB
- Model Layer – Entity classes
